# Real Estate Management System - Frontend

## Overview

A modern web application that allows users to search, browse, and manage real estate listings.

## Features

- Authentication
- Property Search
- Property Details
- Listing Management
- Admin Dashboard

## Technologies

- ReactJS
- Redux Toolkit
- React Query
- React Router
- Ant Design
- Axios
